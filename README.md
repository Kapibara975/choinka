# Projekt choinka - Microsoft MakeCode for micro:bit

> Projekt jest pomyślany o iluminacji choinki.

## Co potrzebne?
> Elementy do micro:bita v1 i v2
* Potencjometr
* Taśma LED (domyślnie 60 ledów)
> Dodatkowe dla v1
* Głośnik

## Użyj jako rozszerzenia

To repozytorium  wykorzystuje **rozszerzenia**, które można od razu zaimportować.

* otwórz [https://makecode.microbit.org/](https://makecode.microbit.org/)
* kliknij na **Nowy Projekt**
* kliknij **Rozszerzenia** w menu oznaczonym kołem zębatym
* szukaj **https://github.com/kapibara975/choinka** i importuj

## Edytuj ten projekt

Aby edytować to repozytorium w MakeCode.

* otwórz [https://makecode.microbit.org/](https://makecode.microbit.org/)
* kliknij **Importuj**, a następnie **Importuj URL**
* wklej '''
  https://github.com/kapibara975/choinka
  ''' i kliknij importuj
